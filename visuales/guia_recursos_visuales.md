# Guia para elegir recursos visuales

## Cuando usar una tabla

Usa tabla cuando necesites comparar datos exactos, por ejemplo produccion por finca.

## Cuando usar grafico de barras

Usa grafico de barras cuando quieras comparar cantidades entre categorias, por ejemplo litros de leche por finca.

## Cuando usar captura de pantalla

Usa captura cuando quieras evidenciar que ejecutaste comandos, corregiste errores o generaste archivos.

## Cuando usar diagrama de flujo

Usa diagrama de flujo para explicar un proceso paso a paso, como lectura de CSV, calculo, reporte y grafico.

## Reglas de calidad visual

- No llenes una diapositiva con parrafos largos.
- Usa titulos claros.
- Cada grafico debe tener una explicacion.
- Muestra solo la informacion necesaria.
- Relaciona cada visual con el problema del reto.
