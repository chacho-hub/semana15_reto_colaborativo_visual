# Checklist de entrega

Antes de subir tu evidencia a Moodle, verifica:

- [ ] Abri el proyecto completo en VS Code.
- [ ] Ejecute `python --version`.
- [ ] Instale dependencias con `pip install -r requirements.txt`.
- [ ] Ejecute `00_verificar_entorno.py`.
- [ ] Ejecute `01_explorar_datos.py`.
- [ ] Corregi los errores de `02_generar_resultados.py`.
- [ ] Corregi los errores de `03_generar_grafico.py`.
- [ ] Ejecute `04_validar_trabajo.py`.
- [ ] Complete la bitacora de acciones.
- [ ] Complete los roles del equipo.
- [ ] Genere reporte y grafico.
- [ ] Complete la plantilla de presentacion.
- [ ] Inclui capturas claras de proceso y resultados.
