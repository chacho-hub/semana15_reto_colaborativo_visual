# Semana 15 - Reto colaborativo y comunicacion visual de resultados

## Proposito del laboratorio

Este proyecto es una practica guiada para trabajar la resolucion de retos colaborativos, la comunicacion de resultados y el uso de recursos visuales dentro del entorno de programacion.

La actividad simula una situacion rural: varias fincas registraron datos de produccion de leche y cosecha de maiz. El equipo debe abrir el proyecto en VS Code, ejecutar scripts, detectar errores, generar resultados, crear recursos visuales y documentar el proceso en una bitacora.

## Que debe lograr el estudiante

Al terminar, el equipo debe entregar evidencias de que pudo:

1. Abrir y navegar el proyecto en VS Code.
2. Ejecutar comandos desde la terminal.
3. Explorar un archivo CSV con datos de produccion rural.
4. Corregir errores intencionales en los scripts.
5. Generar un reporte de resultados.
6. Crear al menos un recurso visual para comunicar los resultados.
7. Registrar roles, acciones, errores y ajustes en la bitacora.
8. Preparar una breve comunicacion de resultados con apoyo visual.

## Primeros comandos sugeridos

Desde la terminal integrada de VS Code ejecuta:

```bash
python --version
pip install -r requirements.txt
python src/00_verificar_entorno.py
python src/01_explorar_datos.py
python src/02_generar_resultados.py
python src/03_generar_grafico.py
python src/04_validar_trabajo.py
```

Importante: los archivos `02_generar_resultados.py` y `03_generar_grafico.py` tienen errores intencionales. No es un fallo del ZIP. Tu tarea es leer los mensajes de error, corregirlos y documentar cada ajuste en `evidencias/bitacora_acciones.md`.

## Entrega esperada

Sube a Moodle una carpeta comprimida o un PDF de evidencias con:

- Capturas del proyecto abierto en VS Code.
- Capturas de comandos ejecutados.
- Bitacora de acciones y errores corregidos.
- Roles del equipo.
- Reporte generado.
- Grafico o recurso visual.
- Plantilla de presentacion completada.
- Reflexion breve sobre el trabajo colaborativo.
